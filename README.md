# 🛡️ CyberAtlas

> **The Ultimate Cybersecurity Learning Roadmap**
>
> Learn Cybersecurity from **Beginner → Advanced** with structured roadmaps, hands-on labs, cheat sheets, tools, scripts, certifications, and real-world resources.

---

## 📖 About

CyberAtlas is an open-source project designed to help students, professionals, ethical hackers, SOC analysts, bug bounty hunters, and penetration testers learn cybersecurity in a structured way.

Instead of searching hundreds of websites, everything is organized into one roadmap.
